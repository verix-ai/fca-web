12 — Guidelines

Brand Usage Rules

✓ Do

  • Use the Navy background (#0c1214) to frame the light Cream (#f4f2ee) application body.
  • Maintain large border-radius (40px) on main layout containers for a soft, friendly feel.
  • Use Mint (#81c567) and Peach (#f4b36d) as primary accent colors.
  • Keep typography strictly to the Outfit font family.
  • Use fully rounded "pill" shapes for buttons.

✗ Don't

  • Don't use sharp, square corners on major layout blocks.
  • Don't overwhelm the user with too many primary colors at once.
  • Don't use script or serif fonts; stay consistent with Outfit.
  • Don't use low-contrast text on the cream or navy backgrounds.
